== Description ==

This plugin will count the number of approved comments each commentator has left on your blog and then reward he/she with a Do-follow link in the Comment Author Link field.

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
